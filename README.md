# Web Development - *Quiz App*

Created by: **Tarina Muna**

This web app: **The Quiz App is a dynamic, browser-based application that lets users test their knowledge through a fun and interactive multiple-choice quiz. Users can choose how many questions they want to answer, and each quiz session randomly selects questions from a larger pool to ensure a fresh experience every time. The app features a clean and colorful design with subtle animations, a countdown timer for added challenge, and a final results screen that displays the user's score.
